Приложение А
Установка программы

После запуска КPA65156_Installer.exe на экран ПК выводится окно установки программы KPA65156.exe, в котором оператору будет предложено выбрать путь для установки необходимых компонентов (см. рисунок А1). Рекомендуется использовать заранее прописанные пути сохранения данных. Для этого при установке достаточно нажать кнопку NEXT. После окончания копирования файлов, установка программного обеспечения завершается нажатием на кнопку FINISH.

Рисунок А1 – Окно установки

Рисунок А2 – Окно завершения установки

Для работы программы и взаимодействия ПК с КПА через USB интерфейс необходимо установить драйвер FTDIChip. Запуск установки драйвера происходит автоматически после установки программы KPA65156.exe.

Рисунок А3 – Окно разархивирования

После запуска установки драйвера на экран ПК выводится окно разархивирования (см. рисунок А3), в котором для распаковки файлов необходимо нажать кнопку EXTRACT. После распаковки файлов выведется окно запуска установки (см. рисунок А4), в нем необходимо нажать кнопку ДАЛЕЕ.

Рисунок А4 – Окно запуска установки

Установка завершается нажатием на кнопку ГОТОВО в окне завершения установки (см. рисунок А5).

Рисунок А5 – Окно завершения установки
