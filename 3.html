3. Выполнение программы

Перед началом работы с программой необходимо собрать рабочее место, изучить инструкцию по проведению испытаний (И18 или ПМ) для проверяемого устройства.

3.1. Запуск программы

Запуск программы KPA65156.exe осуществляется по двойному клику левой кнопкой манипулятора мышь по соответствующей иконке. Запуск программы контролировать по появлению окна представленного на рисунке 1.

Рисунок 1 – Окно ввода исходных данных

Где:
  1. Список ТИП ИСПЫТАНИЙ предназначен для выбора одного из режимов работы программы:
    • Проверка электрической схемы КСР;
    • Проверка работоспособности КСР;
    • Проверка управляющих воздействий;
    • Проверка генерации импульсов;
    • Проверка коммутации;
    • Проверка питания;
    • Ручной режим.
  2. Поле НАИМЕНОВАНИЕ И ЗАВОДСКОЙ НОМЕР ОК предназначено для ввода наименования и заводского номера проверяемого устройства (МУ, ПП, КПА или КСР);
  3. Поле Ф.И.О. ОПЕРАТОРА предназначено для ввода фамилии и инициалов оператора;
  4. Кнопка ПРОДОЛЖИТЬ предназначена для подтверждения ввода исходных данных;
  5. Кнопка ЗАКРЫТЬ предназначена для завершения работы и выхода из программы.

Перед выполнением проверок необходимо в окне ВВОДА ИСХОДНЫХ ДАННЫХ (см. рисунок 1) выбрать из выпадающего списка ТИП ИСПЫТАНИЙ (см. рисунок 1 п.1), ввести сокращенное наименование и номер ОК в поле НАИМЕНОВАНИЕ И ЗАВОДСКОЙ НОМЕР ОК (см. рисунок 1 п.2), вести Ф.И.О оператора в поле Ф.И.О. ОПЕРАТОРА (см. рисунок 1 п.3). После ввода всех данных необходимо нажать кнопку ПРОДОЛЖИТЬ (см. рисунок 1 п. 4) и проконтролировать появление одного из основных окон. Если в окне ВВОДА ИСХОДНЫХ ДАННЫХ в выпадающем списке ТИП ИСПЫТАНИЙ был выбран ручной режим, то контролировать появление окна представленного на рисунке 3, во всех остальных проверках контролировать появление окна представленного на рисунке 2.

Окно программы, представленное на рисунке 2 является окном проверки в автоматическом режиме и появляется при выборе любого режима кроме ручного в окне ВВОДА ИСХОДНЫХ ДАННЫХ (см. рисунок 1).

Рисунок 2 – Окно автоматических проверок

  2. Кнопка НАЧАТЬ ПРОВЕРКУ предназначена для запуска автоматических проверок;

После нажатия программа выполняет последовательно действия в соответствии с инструкцией по проведению испытаний (И18) или методикой испытаний и проверок (ПМ) на проверяемое устройство. Выполнение проверок сопровождается выводом диалоговых окон описанных в разделе 3.4 и сообщений описанных в разделе 3.5 данного руководства.

Во время автоматической проверки в поле СООБЩЕНИЯ ОПЕРАТОРУ (см. рисунок 2 п.1) будет выводиться информация о ходе проверки, а так же производится запись в файл-отчет.

Все сообщения, выводимые в поле СООБЩЕНИЯ ОПЕРАТОРУ (см. рисунок 2 п.1), записываются в файл с именем LOGFILE_<чч.мм.гггг>_<н>.txt (где чч.мм.гггг – дата создания файла, н – номер запуска программы) в папку с именем KPA65156, расположенную в директории МОИ ДОКУМЕНТЫ.

Все сообщения, выводимые в поле СООБЩЕНИЯ ОПЕРАТОРУ (см. рисунок 2 п.1), записываются в файл с именем LOGFILE_<чч.мм.гггг>_<н>.txt (где чч.мм.гггг – дата создания файла, н – номер запуска программы) в папку с именем KPA65156, расположенную в директории МОИ ДОКУМЕНТЫ.

Для завершения работы с программой и закрытия окна автоматических проверок необходимо нажать кнопку ЗАКРЫТЬ (см. рисунок 2 п.5).

3.3. Окно ручного режима работы

Окно программы, представленное на рисунке 3 является окном ручных проверок и появляется при выборе ручного режима в окне ВВОДА ИСХОДНЫХ ДАННЫХ (см. рисунок 1).

Рисунок 2 – Окно автоматических проверок

Где:
  1. Поле СООБЩЕНИЯ ОПЕРАТОРУ, в которое выводятся результаты измерений и действия оператора;
  2. Область ПЕРЕКЛЮЧЕНИЕ КАНАЛОВ предназначена для подключения к мультиметру одной из электрических цепей;
  3. Область ИЗМЕРЕНИЕ предназначена для проведения измерения напряжения или сопротивления электрической цепи и задания способа подключения;
  4. Область УПРАВЛЯЮЩИЕ ВОЗДЕЙСТВИЯ предназначена для коммутации шин питания «+27В», «-27В» от источника питания с помощью переключателей K1...K21;
  5. Кнопка ИМПУЛЬСНОЕ ВОЗДЕЙСТВИЕ предназначена для подачи управляющих воздействий с длительностью менее одной секунды;
  6. Кнопка К5, К6, К11, К12 < 1000 мс предназначена для одновременной подачи управляющих воздействий по каналам К5, К6, К11, К12 для установки прибора КСР в исходное состояние после включения (в соответствии с 17КС.33Ю 2334-O ТУ);
  7. Кнопка ОТКРЫТЬ ФАЙЛ ПРОВЕРКИ предназначена для открытия файла с результатами испытаний или проверки;
  8. Кнопка ЗАКРЫТЬ предназначена для завершения работы и выхода из программы.

Для подключения одного из каналов Y к мультиметру, необходимо выбрать его из выпадающего списка в области ПЕРЕКЛЮЧЕНИЕ КАНАЛОВ (см. рисунок 3 п.2), после нажать кнопку ПЕРЕКЛЮЧИТЬ в той же области. По нажатию на кнопку СБРОСИТЬ в области ПЕРЕКЛЮЧЕНИЕ КАНАЛОВ электрическая цепь отключается от мультиметра. Каналы и соответствующие им электрические цепи прибора, подключаемые к мультиметру, перечислены в таблице Б1 приложения Б. После подключения к мультиметру в поле СООБЩЕНИЯ ОПЕРАТОРУ (см. рисунок 3 п.1) выводится строка – «КАНАЛ Y<номер канала> ПОДКЛЮЧЕН (A:<двоичное число>, B:<двоичное число>, C:<двоичное число>, D:<двоичное число>), ВРЕМЯ: <чч.мм.сс>».

Измерение заданного канала проводится после нажатия на кнопку ИЗМЕРИТЬ в области ИЗМЕРЕНИЕ (см. рисунок 3 п.3). Результат измерения выводится в поле СООБЩЕНИЯ ОПЕРАТОРУ в форме строки – «Измеренное значение <значение> = <величина> <ед.изм.> (<способ подключения>)». В выпадающих списках области ИЗМЕРЕНИЕ (см. рисунок 3 п.3)доступна настройка способа подключения электрической цепи к мультиметру ПРЯМОЕ ПОДКЛЮЧЕНИЕ / ОБРАТНОЕ ПОДКЛЮЧЕНИЕ, и настройка режима измерения ИЗМ.НАПРЯЖЕНИЯ / ИЗМ.СОПРОТИВЛЕНИЯ. При прямом способе подключения к клемме «+» мультиметра подключается контакт прибора соответствующий колонке «+», а клемма «–» коленке «–»таблицы Б1 приложения Б. При обратном способе подключения к клемме «+» мультиметра подключается контакт прибора соответствующий колонке «–», а клемма «–» колонке «+» таблицы Б1 приложения Б.

Включение канала проводится после нажатия кнопки К1…К21 в области УПРАВЛЯЮЩИЕ ВОЗДЕЙСТВИЯ (см. рисунок 3 п.4). После включения переключатель окрашивается ярко жёлтым цветом, а в поле СООБЩЕНИЯ ОПЕРАТОРУ выводится строка – «КАНАЛ Y<номер канала> ПОДКЛЮЧЕН (A:<двоичное число>, B:<двоичное число>, C:<двоичное число>, D:<двоичное число>), ВРЕМЯ: <чч.мм.сс>». Каналы и соответствующие им электрические цепи прибора перечислены в таблице В1 приложения В.

Для подачи импульсного воздействия, необходимо нажать кнопку ИМПУЛЬСНОЕ ВОЗДЕЙСТВИЕ (см. рисунок 3 п.5), проконтролировать появление надписи ВКЛ. и смену цвета кнопки на желтый, после подать нужное воздействие в области УПРАВЛЯЮЩИЕ ВОЗДЕЙСТВИЯ (см. рисунок 3 п.4).

Для выключения режима импульсных воздействий необходимо нажать кнопку ИМПУЛЬСНОЕ ВОЗДЕЙСТВИЕ, проконтролировать появление надписи ВЫКЛ.

Установка прибора КСР в исходное состояние проводится после нажатия кнопки «К5, К6, К11, К12 < 1000 мс» (см. рисунок 3 п.6).

Для просмотра файла-отчета необходимо нажать кнопку ОТКРЫТЬ ФАЙЛ ПРОВЕРКИ (см. рисунок 3 п.7).

Все сообщения, выводимые в поле СООБЩЕНИЯ ОПЕРАТОРУ (см. рисунок 3 п.1), записываются в файл с именем LOGFILE_<чч.мм.гггг>_<н>.txt (где чч.мм.гггг – дата создания файла, н – номер запуска программы) в папку с именем KPA65156, расположенную в директории МОИ ДОКУМЕНТЫ.

Для завершения работы с программой и закрытия окна ручной проверки необходимо нажать кнопку ЗАКРЫТЬ (см. рисунок 3 п.8).

3.5. Сообщения оператору

В данном разделе руководства оператора описаны сообщений оператору выводимые в процессе работы с программой.

3.5.1 Нештатные ситуации

В процессе работы с программой при проведении испытаний оператор осуществляет контроль над оборудованием через информационные окна программы и управляет испытаниями непосредственно через основное и диалоговые окна программы. При возникновении нештатной ситуации в работе с программой, например, недостаточного места на диске, нехватки памяти или отсутствии канала связи, оператору могут быть выданы сообщения через операционную систему, при этом необходимо прервать выполнение текущей проверки и разобраться в выданном сообщении.

3.5.2. Сообщения при проверке электрической схемы КСР

Проверка электрической схемы прибора проводится в соответствии с 17КС.33Ю 2334-0 ПМ. В результате выполнения проверки программой формируется файл-отчёт с сообщениями, выводимыми в процессе работы программы.

Когда полученный результат измерения не соответствует техническим требованиям, выводится сообщение «!!!ВНЕ ДОПУСКА!!!» в поле СООБЩЕНИЯ ОПЕРАТОРУ. На этом шаге проверка прерывается, а в поле СООБЩЕНИЯ ОПЕРАТОРУ выводится сообщение «ПРОВЕРКА ЭЛЕКТРИЧЕСКОЙ СХЕМЫ ПРЕРВАНА».

3.5.3. Сообщения при проверке работоспособности КСР

Проверка работоспособности прибора проводится в соответствии с 17КС.33Ю 2334-0 ПМ. В результате выполнения проверки программой формируется файл-отчёт с сообщениями, выводимыми в процессе работы программы.

Когда полученный результат измерения не соответствует техническим требованиям, выводится сообщение «!!! ВНИМАНИЕ !!! СОПРОТИВЛЕНИЕ ВНЕ ДОПУСКА» в поле СООБЩЕНИЯ ОПЕРАТОРУ. На этом шаге проверка прерывается, а в поле СООБЩЕНИЯ ОПЕРАТОРУ выводится сообщение «ПРОВЕРКА РАБОТОСПОСОБНОСТИ ПРЕРВАНА».

3.5.4. Сообщения при проверке управляющих воздействий

Проверка управляющих воздействий проводится с модулем управления МУ в соответствии с инструкцией КИНД.441546.388 И18. В результате выполнения проверки программой формируется файл-отчёт с сообщениями, выводимыми в процессе работы программы.

Когда полученный результат при проверке канала не соответствует техническим требованиям, выводится сообщение «КАНАЛ Y<номер> НЕ СООТВЕТСТВУЕТ ТРЕБОВАНИЯМ» в поле СООБЩЕНИЯ ОПЕРАТОРУ.

3.5.5. Сообщения при проверке цепей питания

Проверка цепей питания проводится с пультом проверки ПП в соответствии с инструкцией КИНД.441546.390 И18 и с аппаратурой КПА в соответствии с инструкцией КИНД.441465.156 И18. В результате выполнения проверки программой формируется файл-отчёт с сообщениями, выводимыми в процессе работы программы.

3.5.6. Сообщения при проверке коммутации

Проверка коммутации проводится с пультом проверки ПП в соответствии с инструкцией КИНД.441546.390 И18 и с аппаратурой КПА в соответствии с инструкцией КИНД.441465.156 И18. В результате выполнения проверки программой формируется файл-отчёт с сообщениями, выводимыми в процессе работы программы.

Когда полученный результат при проверке канала не соответствует техническим требованиям, выводится сообщение «КАНАЛ Y<номер> НЕ СООТВЕТСТВУЕТ ТРЕБОВАНИЯМ» в поле СООБЩЕНИЯ ОПЕРАТОРУ.
